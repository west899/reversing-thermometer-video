# 项目名称
颠倒温度计原理与使用 - 60秒中文科普视频

# 项目目标
制作一条 60 秒左右的中文科普视频，面向零基础学生，讲清楚颠倒温度计的原理、使用方式和历史意义。

# 目标观众
- 零基础学生
- 海洋科学/仪器原理入门观众
- 需要快速理解该仪器的人

# 成片要求
- 时长：50~70秒
- 语言：中文普通话
- 成片比例：16:9
- 风格：学术、清晰、克制、可视化强
- 类型：旁白讲解 + 图片/示意图 + 少量AI动画
- 输出：MP4，1080p

# 必须讲到的内容
1. 为什么普通温度计不适合直接测深海温度
2. 颠倒温度计的基本结构：玻璃温度计、汞柱、细颈/狭窄部位
3. 工作原理：在目标深度被机械触发翻转，汞柱在细颈处断开，读数被锁定
4. 典型使用方式：与 Nansen bottle 配合，在指定深度翻转并同时取样/测温
5. 历史意义：曾广泛用于海洋观测，后来被 CTD 等现代设备替代

# 禁止事项
1. 不要把它画成现代电子温度计
2. 不要使用科幻化机械结构
3. 不要错误表达为“倒过来就更精确”
4. 不要出现与主题无关的人物剧情
5. 不要把重点讲成“实时电子传感器”

# 视觉要求
- 优先真实仪器照片、历史海洋仪器图、结构示意图、流程图
- AI生成镜头只作为补充，不作为唯一可信信息源
- 重点视觉关键词：
  - 深海
  - 翻转
  - 细颈
  - 汞柱断开
  - 锁定读数
  - Nansen bottle
  - 历史仪器 vs 现代 CTD

- 成片应能被用于课堂展示
